     ^                               (grippy indicators)
```

### Toolbar Button States

```
[📁 Import]     Normal
[📁 Import]     Hover (slight elevation)
[📁 Import]     Pressed (scaled down)
[📁 Import]     Disabled (faded, 50% opacity)
```

```

## Animation States

### Clip Drop Animation

```
1. Dragging over timeline:
   
   ─────────────────
   [═════] (highlighted track)
   ─────────────────

2. Dropped:
   
   ─────────────────
   [└─┐ Clipping... ┐─┘]  (fade in)
   ─────────────────

3. Final:
   
   ─────────────────
   [════════════════]  (solid clip)
   ─────────────────
```

### Trim Handle Interaction

```
Normal:  [═]                   [═]
         Left handle           Right handle

Dragging: [◄]                   [►]
         (highlighted, enlarged)
```

## Color Coding Reference

```
Timeline Clips:
- Blue (#4F6DF7): Default clip color
- Purple (#7B8AE6): Selected clip
- Red (#FF6B6B): Playhead indicator
- Gray (#252525): Track background
- Dark (#151515): Timeline background

Semantic Colors:
- Green (#4CAF50): Success
- Orange (#FFA726): Warning
- Red (#EF5350): Error
- Blue (#42A5F5): Info
```
